# 📰 News App

A modern Android News App built using Kotlin and MVVM architecture.  
The app fetches the latest news from a remote API and displays articles in a clean, user-friendly interface.

## Features

- 🔍 Search news by keyword
- 📰 Latest headlines from API
- 📱 Responsive and clean UI
- ⚡ Fast loading with RecyclerView
- 🏗️ MVVM Architecture
- 🌐 Retrofit for API integration
- 🔄 Kotlin Coroutines & Flow
- 💉 Dependency Injection with Hilt
